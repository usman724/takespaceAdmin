Admin API: Teacher Analytics (asteacher-analytics)

Endpoint
- Method: GET
- Path: /api/v1/admin/teacher-analytics/{teacher_id}/
- Auth: Bearer JWT (Authorization: Bearer <token>)
- Tags: Admin

Path Parameters
- teacher_id (integer, required): The teacher's ID.

Query Parameters
- date_range (string, optional, default: 30d): Accepted values: 30d, 7d, all_time, last_year, today, yesterday.
- grade (string, optional): Grade ID filter.
- subject (string, optional): Subject ID filter.

Success Response (200)
- data.homework_questions
  - labels: string[] (student names)
  - data: number[] (counts per student)
  - goal: number
  - avg: number
- data.classroom_questions
  - labels: string[] (student names)
  - data: number[] (counts per student)
  - goal: number
  - avg: number
- data.homework_to_classwork_questions
  - classwork: number
  - homework: number
- data.homework_to_classwork_time
  - classwork_time: number
  - homework_time: number
- data.teacher_engagement: number (percentage)
- data.difficult_topic_leaderboard: Array<{
  - subject_name: string
  - topics: Array<{
    - code: string
    - name: string
    - student_names: string[]
  }>
}>

Example Request
curl 'https://dev.takespace.com/api/v1/admin/teacher-analytics/10/?date_range=30d&grade=2&subject=5' \
  -H 'Authorization: Bearer <JWT>' \
  -H 'accept: application/json'
